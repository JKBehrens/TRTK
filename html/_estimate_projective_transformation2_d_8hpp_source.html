00427     Explanation of the algorithm:
00428 
00429     By using homogeneous coordinates an projective transformation can be described as follows:
00430 
00431         | a_11  a_12  a_13 |     | source_x |    | target_x' |
00432         | a_21  a_22  a_23 |  *  | source_y | =  | target_y' |

00434

00436 
00437         | target_x |     | target_x' / target_z' |
00438         | target_y |  =  | target_y' / target_z' |
00439         | target_z |     | 1                     |.
00440 
00441     The matrix A can be normalized such that component a_33 is equal to one, i.e. A := A' / a'_33, because of the
00442     subsequent normalisation step. Hence A has only 8 DoF. Since the normalisation step is nonlinear, we use
00443     Newton's method (after Simpson) to solve for the entries a_11, ..., a_32.
00444 
00445     The equation can be rewritten as a function f of the sought variables a_11, ..., a_32:

00448         f  =  |     |  =  --------------------------------------  *  |                                          |
00449               | f_2 |      a_31 * source_x + a_32 * source_y + 1     | a_21 * source_x + a_22 * source_y + a_23 |.
00450 
00451     Then we try to find the zero-crossing of the residual (and we are done):
00452 
00453         residual(a_11, ..., a_32) := f(a_11, ..., a_32) - |target_x, target_y|^T.
00454 
00455     This can be done iteratively by computing
00456 
00457         A_new = A_old - Jacobian(residual)^(-1) * residual

00462         Jacobian = |                                                                                                              |
00463                    | 0              0              0     c_3*source_x   c_3*source_y   c_3   -c_2*c_4*source_x  -c_2*c_4*source_y |

00477 
00478         delta_A * Jacobian = -residual.

00481 
00482         delta_A = - (Jacobian^T * Jacobian)^(-1) * (Jacobian^T * residual).
00483 
00484     Then (in each iteration step) the new solution vector becomes:
00485
